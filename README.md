# SmartGreenhouse
智能大棚(物联网大课设)

## 技术选型

| 功能端 |                       技术描述                       |
| :----: | :--------------------------------------------------: |
|  安卓  |                    flutter, dart                     |
|  前端  | vue, typescript, element-ui, vuex, vue-element-admin |
|  后端  |         python, flask, mysql, redis, docker          |
|  硬件  |              raspberry pi, python, emqx              |



## 已知问题

| 功能端 |                           问题描述                           |
| :----: | :----------------------------------------------------------: |
|  安卓  |               1. 过于简单，没有时间查询等功能                |
|  前端  |                1. 查询方面寻找更好的显示方案                 |
|  后端  | 1. 由于部分字段不能用字符串查询所以用了拼接字符串，存在注入风险 2.jwt部分没有做黑名单处理，存在信息盗用风险 |
|  硬件  |          1.缺少开关等互动装置 2.缺少灯管等反馈装置           |
